## Commands

### Pre-load

1. ``npm install express-generator -g``
2. ``cd SWC``
3. ``npm install``
4. ``npm audit fix --force``
5. ``DEBUG=myapp:* npm start``

3. ``git add .``
4. ``git commit -m "<수정한 내용>" ``
5. ``git push``
